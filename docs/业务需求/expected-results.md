# SQL 模板预期结果样例（基于 v1.3 真实参考数据）

## 1. 用途

这份文档给本地 / demo / 回归测试的 TiDB v1.3 seed 配一套人工对账用预期结果，用于校验当前已产品化的核心 SQL 模板：T01 / T02 / T03 / T04 / T06 / T08 / T09 / T10 / T11 / T12 / T13。

当前默认对账窗口已经切换为 `docs/业务需求/seed_data_refer/` 参考导出生成的真实数据，不再默认依赖 `990001 / 990011` 固定夹具。旧 `seed_data_local/regression_fixture.sql` 仅作为 legacy/专项固定夹具保留，需要时显式导入。

完整 CSV 对账表见：`docs/业务需求/csv/00_测试参数与总校验点.csv` ~ `docs/业务需求/csv/11_T13_首存金额分桶.csv`。
